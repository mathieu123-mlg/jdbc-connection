# JDBC Connection (example)

This repository provides a JDBC connection example between Java and PostgreSQL.

## Important Note
**`Please verify your environment variables`** to ensure a successful connection.

## Configuration
Update the following variables in the code (lines 8-10) with your actual PostgreSQL credentials:

- **PORT**: 5432 → `your_PORT`
- **DB**: Database → `your_DATABASE`
- **password**: password → `your_PASSWORD`

## Prerequisites
- Java Development Kit (JDK)
- PostgreSQL database
- PostgreSQL JDBC driver

## Usage
1. Clone this repository
2. Update the connection variables with your database credentials
3. Run the Java application
